## Инструкция по локальному запуску:
* Клонировать репозиторий:
	- ```git clone https://github.com/ponomarenkoboris/chat.git```
	- ```cd chat```
* Установить зависимости, необходимые для работы приложения:
	- ```yarn```
* Запуск приложения:
	- ```yarn dev```
